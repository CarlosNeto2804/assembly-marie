# Desafio - 1 
- Fazer um programa em linguagem de montagem **MARIE** que some uma constante a cada elemento de um vetor de 10 elementos. 
- Guardar o valor da soma de cada **elemento+constante** no respectivo elemento do vetor.
 
## Exemplo
```
constante: 10
vetor: [20, 30, 40, 50]
execução da soma
vetor: [30, 40, 50, 60]
```
